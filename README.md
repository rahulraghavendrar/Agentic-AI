# Agentic AI Projects

This repository contains my hands-on implementations while learning Agentic AI.

---

## Projects

### Employee Details Agent

Status: ✅ Completed

Description:

Built an AI agent that extracts employee information from an unstructured PDF document and converts it into a structured JSON format using the Gemini API.

Pipeline:

Employee PDF
↓
PDF Loader
↓
Text Extraction
↓
Gemini
↓
Structured JSON

Libraries Used:

- Google Gemini API
- LangChain Community (PyPDFLoader)
- Python Dotenv

Output:

- Employee Name
- Employee ID
- Department
- Designation
- Salary
- Joining Date
- Manager
- Email
- Phone
- Location
